# 高质量网站资源备忘录

> 整理一些长期可用、权威、实用的高质量网站，涵盖开发、学习、设计、工具等方向，便于日常查阅与知识沉淀。

---

## 🧑‍💻 开发者资源

- **[MDN Web Docs](https://developer.mozilla.org)**  
  Mozilla 官方 Web 技术文档（HTML/CSS/JavaScript/API），前端开发必备。

- **[Stack Overflow](https://stackoverflow.com)**  
  全球最大的程序员问答社区，解决编码难题首选。

- **[GitHub](https://github.com)**  
  开源代码托管与协作平台，全球开发者聚集地。

- **[GitLab](https://gitlab.com)**  
  类似 GitHub 的 DevOps 平台，支持私有仓库与 CI/CD。

- **[DevDocs](https://devdocs.io)**  
  聚合多语言 API 文档（React、Vue、Python、Go 等），支持离线与快速搜索。

- **[The Algorithms](https://github.com/TheAlgorithms)**  
  各编程语言实现的经典算法与数据结构开源项目。

## 📚 学习与知识

- **[freeCodeCamp](https://www.freecodecamp.org)**  
  免费编程学习平台，含 HTML/CSS/JS、算法、数据可视化等实战项目。

- **[Khan Academy](https://www.khanacademy.org)**  
  免费数学、计算机科学、经济、人文等课程，适合系统性学习。

- **[Coursera](https://www.coursera.org)**  
  与斯坦福、耶鲁等名校合作的在线课程平台（部分课程免费）。

- **[MIT OpenCourseWare](https://ocw.mit.edu)**  
  麻省理工学院公开课程资源，涵盖计算机、物理、工程等领域。

- **[Wikipedia](https://www.wikipedia.org)**  
  全球最大百科全书（建议交叉验证信息准确性）。

## 🎨 设计与 UI/UX

- **[Material Design](https://m3.material.io)**  
  Google 官方设计语言与组件库（Material 3）。

- **[Apple Human Interface Guidelines](https://developer.apple.com/design)**  
  Apple 官方人机交互设计规范，iOS/macOS 应用开发参考。

- **[Dribbble](https://dribbble.com)**  
  设计师作品展示平台，获取 UI/UX 灵感。

- **[Figma Community](https://www.figma.com/community)**  
  免费可复用的 UI 组件、模板与设计系统。

- **[Coolors](https://coolors.co)**  
  一键生成配色方案，支持导出与调色板管理。

## ⚙️ 工具与效率

- **[Notion](https://www.notion.so)**  
  强大的笔记、数据库与知识管理工具。

- **[Obsidian](https://obsidian.md)**  
  本地 Markdown 笔记应用，支持双向链接与知识图谱。

- **[Carbon](https://carbon.now.sh)**  
  美化代码截图，支持主题、字体、背景自定义。

- **[JSON Formatter](https://jsonformatter.org)**  
  JSON 格式化、压缩、校验与高亮工具。

- **[Regex101](https://regex101.com)**  
  正则表达式在线测试、解释与调试平台。

- **[Awesome Lists](https://github.com/sindresorhus/awesome)**
